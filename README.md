# 오운완
## 프로젝트 소개
> 오운완 프로젝트는 스포츠 과학 분야에 사용하는 EMG 기기를 대중화해 운동을 보다 과학적으로 하고 측정된 값들과
운동 지식을 서로 공유하는 커뮤니티를 통해 운동을 도와주는 헬스 플랫폼입니다.
